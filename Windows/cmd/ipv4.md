## ipv4.cmd

A simpler way to get the current ipv4 address of the machine that the command is executed on.

Calls `ipconfig` and pipes it to `findstr /i 'ipv4`, then performs basic text extraction to get the address.

Accepts no parameters.

### Examples:

```bash
C:\>ipv4
192.168.1.220
```

## ipv4_qr.cmd

Prints out a qr code to the address of the local machine's `ipv4` address.

Calls `curl` to `qrenco.de/{http(s)}://{host}:{port}`

### Accepts parameters:

- `-sec` / `-s` `<boolean>` : "Secure"; whether to use https or http. Defaults to http.
- `-port` / `-p` `<integer>`: "Port"; the port of the address to send to `qrenco.de`. Defaults to 80.
- `-verbose` / `-v` `<boolean>`: "Verbose"; whether to print out info about Secure, host, and port.
